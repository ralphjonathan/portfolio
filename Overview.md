## 6. Proposed Technical Specifications & Design (AI Suggestions)

### Design & Aesthetics
* **Vibe**: "Modern Tech & Precision Engineering"
* **Theme**: Dark Mode by default. Deep backgrounds with high-contrast glowing elements to mimic precision instruments and software terminals.
* **Color Palette**: 
  * Background: Jet Black or Deep Slate (`#0B0F19`)
  * Text: Off-white/Silver (`#E2E8F0`)
  * Accents: Electric Cyan (`#00F0FF`) or Neon Green (`#10B981`) for interactive elements, glowing buttons, and graph bars.
* **Typography**:
  * Headings/Base: Clean, modern Sans-Serif (e.g., *Inter*, *Roboto*, or *Outfit*).
  * Data/Code/Subtitles: Technical Monospace (e.g., *JetBrains Mono* or *Fira Code*) to emphasize the engineering/programming vibe.
* **Animations**: Smooth micro-interactions. A satisfying loading animation (perhaps a glowing circuit line), hover effects on buttons, and fade-ins on scroll.

### Tech Stack
* **Core**: HTML, CSS, JavaScript.
* **Framework**: React.js (via Vite) for a dynamic, component-based, single-page application (SPA) experience. It makes carousels and animated graphs easier to build.
* **Styling**: Vanilla CSS with custom properties or Tailwind CSS for rapid styling that aligns with our aesthetic.
* **Animations**: Framer Motion (React) for the animated bar graph and page transitions.
* **Deployment**: Vercel or GitHub Pages (perfect for portfolio hosting).
